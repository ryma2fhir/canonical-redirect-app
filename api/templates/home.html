Canonical Redirect App

This is an example application to show how to use the Simplifier.net Resolve URL to make your FHIR canonicals resolve in the browser.

  • Try an direct redirect example: {{example_indirect_canonical}}
  • Try an indirect redirect example: {{example_direct_canonical}}

Both URLs will redirect you to https://simplifier.net/resolve with the appropriate canonical and scope parameter. From there Simplifier will directly redirect you immediately to the resource page for that canonical within the corresponding FHIR project or package.
